## What is Data?
Data is information or facts collected about something. It can be numbers, words, pictures, measurements, or observations.

**Examples:** 
- Age of students in a class
- Sales of products in a shop
- Daily temperatures

---

## What is Data Analysis?
Data analysis is the process of inspecting, cleaning, and organizing data to find useful information and support decisions.

**Example:** 
- A shop can look at sales data to know which product sells the most and when

---

## Key Differences

### Data Science vs Data Analysis
- **Data Science:** A broader field that uses data to make predictions, build models, and solve complex problems. Often uses programming, machine learning, and statistics.  
- **Data Analysis:** A smaller part of data science, focused on inspecting, cleaning, and interpreting data to find insights or trends.

**Example:**  
- Data analysis â†’ Checking sales data to see which product sells most  
- Data science â†’ Using sales data to predict future sales and trends

### Data Analysis vs Data Analytics
- **Data Analysis:** The process of examining data to find useful information and insights.  
- **Data Analytics:** The application of data analysis to solve specific business problems. It is more action-oriented.

**Example:**  
- Data analysis â†’ Calculating the average daily sales  
- Data analytics â†’ Using those insights to decide on promotions or inventory management

## Introduction
Not all data is the same. In data analysis, understanding the type of data you are working with is very important because it determines how you analyze and interpret it.

---

## 1. Qualitative Data (Categorical Data)

Qualitative data describes qualities or characteristics. It is non-numerical and cannot be measured with numbers.

Examples:
- Gender (Male/Female)
- Eye color
- Country
- Marital status
- Product category

This type of data is grouped into categories.

---

## 2. Quantitative Data (Numerical Data)

Quantitative data consists of numbers and can be measured or counted.

Examples:
- Age
- Height
- Salary
- Number of students
- Sales revenue

Quantitative data can be analyzed using mathematical operations.

---
